# Vertical Jump Program — Progress Tracking

## Testing Protocol

- **Verticals:** Tested in Week 8 (Realization phase). Record peak jump of the session. Method: video against backboard with ruler — peak touch height recorded.
- **Back Squat:** Track 5RM each cycle as the primary strength marker. Estimated 1RM calculated via Epley. No true max test until Cycle 6.
- **Power Clean:** Record top working set (weight × reps) each cycle. True max test only at Cycle 6 moonshot.
- **Bodyweight:** Recorded on test day, morning, fasted.
