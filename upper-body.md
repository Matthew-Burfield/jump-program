# Upper Body Program

Supplementary upper body work — kept separate from jump program to avoid interference with leg/vert focus.

## Progression Rule

- Log RPE for any set that feels hard; always log the last set
- Increase weight when the last set finishes at **RPE 8 or below across 2 consecutive sessions**
- Increment: **1.25kg** per increase
- Rep completion trend (fewer failures at same weight) counts as progress — don't only track weight on the bar
